# Workbench

Workbench is a free Unreal Engine plugin that streamlines everyday workflows for cinematic, previs, and virtual production artists.

---

## Supported Unreal Engine Versions

-  Unreal Engine 5.6.1

---

## Features

###  Camera Ripper
- Import Camera FBX
- Create Image Plate
- Image Plate Visibility
- Image Plate Opacity

###  Measurement Tool
- Click-to-Measure
- Save Measurement
- Unit Conversion

###  Utilities
- Sequencer Visibility
- Length Converter

---

## Installation

1. Download the latest Release.
2. Extract the ZIP.
3. Copy the **Workbench** folder into your project's **Plugins** folder.
4. Restart Unreal Engine.
5. Launch the Workbench Editor Utility Widget.
